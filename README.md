# trilha-serpentina-numerada
O tabuleiro agora é uma trilha serpentina numerada (estilo da imagem): 
linhas de 5-7 casas que viram em zigue-zague de baixo pra cima, 
com uma "estrada" SVG conectando as casas em sequência 
e setas curvas laranja indicando os atalhos das bifurcações.
A cada fase, o layout varia ligeiramente: número de colunas (5-7) 
e linhas (4-6) mudam, alterando o tamanho e o formato da serpentina, 
mantendo toda a lógica (dado, drag do token, 
bônus/moeda/armadilha/fork, vidas, loja de power-ups).
